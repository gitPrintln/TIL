## TCP 3-Way Handshake

#### 핵심 개념
```js
TCP 연결을 시작할 때 서로 신뢰 가능한 연결을 만드는 과정
연결 과정 (3단계)
```
```js
1️⃣ SYN
클라이언트 → 서버
“연결할래?”

2️⃣ SYN-ACK
서버 → 클라이언트
“그래, 나 준비됐어”

3️⃣ ACK
클라이언트 → 서버
“확인, 연결 시작”
➡ 이 단계가 끝나면 정상 통신 시작
```
- 보안과의 관계
SYN Flood 공격
ACK 안 보내고 SYN만 계속 보내서 서버 자원 고갈

#### 방어
- SYN Cookie
- 방화벽/IPS 설정

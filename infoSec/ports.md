## Well-Known Ports (주요 포트 번호)
```js
핵심 개념

서비스마다 기본으로 사용하는 포트 번호
→ 시험에서 “이 서비스는 몇 번 포트?” 단골 출제

- 자주 나오는 포트
20 / 21 : FTP (데이터 / 제어)
22 : SSH
23 : Telnet ❌ (보안 취약)
25 : SMTP (메일 발송)
53 : DNS
80 : HTTP
110 : POP3
143 : IMAP
443 : HTTPS
```
```js
보안 포인트
Telnet(23) → SSH(22) 로 대체
불필요한 포트는 차단
열린 포트 = 공격 표면 증가
```

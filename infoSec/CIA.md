## CIA 3요소 (기밀성·무결성·가용성)

- 정보보안의 근본 원칙이자 모든 문제·이론·기술의 출발점.

#### 1) 기밀성(Confidentiality)
- “볼 사람이 아닌 사람은 절대 못 보게 한다.”
```js
대표 기술

암호화(AES, RSA)

접근통제(ACL, RBAC)

인증(Password, OTP 등)

ex)
데이터베이스를 암호화해서 내부자라도 못 보게 함

HTTPS로 모니터링해도 내용 못 읽음
```

#### 2) 무결성(Integrity)
- “내용이 몰래 바뀌지 않도록 한다.”
```js
대표 기술

해시(SHA-256)

디지털 서명

체크섬

MAC(Message Authentication Code)

ex) 
파일 다운로드 후 SHA-256 값 비교

전자문서에 서명해서 수정 여부 검증
```

#### 3) 가용성(Availability)
- “필요할 때 정상적으로 사용할 수 있어야 한다.”
```js
대표 기술/대응

이중화(서버, 네트워크)

백업/복구

장애조치(HA)

DDoS 방어

ex)
은행 서비스가 장애 없이 24시간 동작

공격받아도 우회경로로 서비스 유지
```
